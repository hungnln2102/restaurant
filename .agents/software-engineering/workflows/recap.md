## /recap — Software Engineering Recap

1. Liệt kê các thay đổi đã thực hiện (files modified, features added, bugs fixed).
2. Trạng thái từng hạng mục: ✅ Done / 🔄 In Progress / ⏳ Pending / ❌ Blocked.
3. Các test đã pass/fail.
4. Technical debt hoặc TODO còn lại.
5. Bước tiếp theo theo thứ tự ưu tiên.
