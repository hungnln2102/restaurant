## /seo_audit — SEO Audit

1. **Technical SEO**:
   - [ ] Sitemap.xml tồn tại và được cập nhật?
   - [ ] Robots.txt đúng cấu hình?
   - [ ] Canonical tags đúng chưa?
   - [ ] Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1?
   - [ ] HTTPS enabled?
2. **On-page SEO**:
   - [ ] Title tag: 50–60 ký tự, có keyword chính?
   - [ ] Meta description: 150–160 ký tự, có CTA?
   - [ ] H1 duy nhất mỗi trang?
   - [ ] Alt text cho ảnh?
   - [ ] Internal linking hợp lý?
3. **Structured Data**: Schema.org đã implement (Product, BreadcrumbList, FAQPage)?
4. Tổng hợp danh sách vấn đề theo mức ưu tiên.
